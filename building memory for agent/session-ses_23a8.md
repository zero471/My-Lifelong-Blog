### 1. 框架/主题对比

*   **Hugo + CaiJimmy/hugo-theme-stack**
    *   **特点**：极其优秀的卡片式设计，左侧边栏+右侧内容的经典布局。开箱即用，非常适合做生活类的归档。

*   **Hexo + Butterfly / Fluid**
    *   **特点**：在国内开发者中极其流行，功能大而全，集成了各种评论、插件。

*   **Next.js + timlrx/tailwind-nextjs-starter-blog**
    *   **特点**：基于 React 和 Tailwind CSS。它不仅是一个博客，更是一个现代化的 Web 应用底座。
    *   **优点（推荐理由）**：
        1.  **极高的设计自由度**：Tailwind 使我们可以轻松覆写样式，打造专属的排版和色彩美学（比如做成高端杂志风、极简自然风）。
        2.  **完美的交互扩展性**：因为是 React 生态，实现“旅行地图可视化”会变得极其简单（可以直接引入 `react-simple-maps`，`framer-motion` 做各种炫酷的旅行轨迹动画和照片瀑布流）。
        3.  **MDX 支持**：你可以在 Markdown 里直接写 React 组件。比如在分享“薅羊毛”经验时，直接插入一个漂亮的自定义卡片组件；在“厨艺”文章里插入一个动态的食谱组件。
