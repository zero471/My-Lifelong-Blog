# My Lifelong Blog 项目进度与状态报告

本文档记录了个人生活博客项目的总体目标、当前开发进展以及未解决的问题与待办事项。

## 🏛 项目总体目标

搭建一个**个人生活博客网站** —— **"My Lifelong Blog"**，与学术无关，专注于记录生活、经验和回忆。

### 版权声明
- 网站页脚必须标明：`© 2026 TMYZ. All rights reserved. Content may not be reproduced without permission.`

### 核心模块 (6大板块)

| 板块 | 说明 |
|------|------|
| 🧠 思考 (Thoughts) | 人生哲思、随笔 |
| 💡 生活经验 (Experiences) | 薅羊毛、邀请链接等实用技巧 |
| 🛒 购物指北 (Goods) | 好物推荐 |
| ✈️ 旅行 (Travel) | **交互式地图** + 照片墙 + 城市记忆 |
| 🍳 厨艺 (Cooking) | 厨艺记录 |
| 💪 运动健身 (Fitness) | 健身经验 |

### 技术选型 & 设计方向
- **框架**: `timlrx/tailwind-nextjs-starter-blog` (Next.js + Tailwind CSS v4 + MDX)
- **动画**: `framer-motion` (拍立得物理交互)
- **地图**: `react-simple-maps` + `d3-geo` (旅行页)
- **美学风格**: 杂志画报与自然原质风 (Editorial & Natural)
  - 色彩: 米白 `#F9F8F6` / 鼠尾草绿 / 陶土红 `#C17767`
  - 字体: `Newsreader`(衬线标题) + `Inter`(正文)
  - 全局纸质噪点纹理

## ✅ 目前项目进度与进展

目前项目已经完成了基础架构搭建、全局视觉语言重构，以及首页和核心的旅行地图页初步开发，主要包括：

1. **项目初始化与美学重构**: 
   - 注入了完整的原质风调色板、自定义双字体（Inter + Newsreader）和全局噪点背景。
   - `data/siteMetadata.js` 已更新（标题 "My Lifelong Blog"，语言 `zh-CN`，亮色主题）。
2. **首页开发 (`/page.tsx`)**: 
   - 移除了默认的博客文章列表，改写为极简介绍风格。
   - 实现并接入 `PolaroidGrid` (拍立得网格) 组件，带有交错入场动画和交互式物理 hover 效果。
   - 头部 (`Header.tsx`) 透明化适配。
3. **性能与 Bug 修复**: 
   - 彻底修复了由于 `MobileNav` 和 `ThemeSwitch` 中 Headless UI Portal 造成的 **Hydration Error**。
   - 解决了 `framer-motion` 在 TypeScript 下的 `Variants` 类型兼容报错，优化了卡片动画性能。
   - 修复了本地开发因 Next.js 缓存损坏导致的 500 错误循环问题。
4. **旅行页面初步完成 (`/travel`)**:
   - 构建了暖色调的 SVG 交互式世界地图组件 (`TravelMap.tsx`)。
   - 足迹点带有陶土红脉冲光晕，支持悬停放大。
   - 实现了点击足迹后弹出带毛玻璃遮罩的**侧边详情面板**（包含城市信息、拍立得风格照片墙）。
5. **近期完成的优化与修复**:
   - **旅行地图体验满级优化**: 
     - 彻底修复了滚轮/触控板上下滑动与页面滚动冲突的问题，同时兼顾了精准缩放；
     - 通过延迟挂载（`isMounted`）彻底消灭了坐标浮点精度导致的 SSR Hydration Error；
     - **黑科技无限循环地图**：通过数学计算克隆 5 个 SVG 地图拼接，并修改 D3 `translateExtent` 像素界限，实现了肉眼无缝的左右无限拖拽以及完美弹性的边界阻挡。不仅南北极完整可见，同时还能保持 `hover` 等事件在所有克隆地图间的“量子同步”，数据维护保持完全解耦。
   - **详情面板升级**: 侧边栏加宽至 `max-w-xl` 以优化阅读体验，并实现了中国大陆与台湾的悬停高亮联动。
   - **Logo 与 Favicon 替换**: 网站头部 Logo 以及全部尺寸的浏览器标签页图标均已替换为自定义图像。
   - **全站子页面模块落地**: 成功重构并上线了**思考、生活经验、购物指北、厨艺、运动健身**这 5 个核心板块的内容落地页。采用了具备原质风与大图卡片元素的响应式瀑布流（Masonry）排版，并撰写了相应的测试 MDX 文章以验证全站路由和渲染逻辑的闭环。
   - **内容与结构清理**: 移除了脚手架自带的冗余文章模板及无用组件（如 `app/Main.tsx`），并完成了 `About Me` 页面的个人专属定制（Jensen Huang, SJTU PhD, Video Understanding/VLM, 福建龙岩）。
